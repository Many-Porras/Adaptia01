Belcorp

Client: Belcorp

Digital Catalog





"El catálogo digital de Belcorp brinda al usuario la experiencia de visualizar productos en un catálogo interactivo. Además, esta plataforma permite realizar pedidos de productos a través de una consultora de belleza, cerrando el círculo de experiencia y fortaleciendo el vínculo entre cliente y marca."

"Desarrollamos esta herramienta, bajo el enfoque de agilidad Scrum, para las marcas Ésika, Cyzone y L’bel aplicando analítica UX y UI para crear una experiencia única y usababilidad amigable para los usuario, lo cual se ve reflejado en los logros en ventas de la compañía."
